Etapes du TP => hackathon :

  1. partir du fichier Excel pour faire une base de données (fichier classeur sous teams ou dans document)
  2. Faire un formulaire HTML qui demande le nom d'utilisateur (google suggest) et renvoie toutes les informations de l'utilisateur
  3. Faire une interface web pour permettre aux utilisateurs d'ajouter un mot de passe et leurs skills (langages préférés, compte Git)
  4. Faire une interface pour composer son équipe (requête de demande d'intégration => envoie de messages entre étudiants)
  5. gérer l'activation et la désactivation des comptes pour gérer les années
